Interactive Web Calculator
A clean, responsive, and functional calculator web application built using HTML5, CSS3 (Grid Layout), and Vanilla JavaScript. This project demonstrates DOM manipulation, event handling, and modern UI design principles.

🚀 Features
Basic Arithmetic: Supports addition, subtraction, multiplication, and division.

Real-time Display: Dynamic updating of the input field as buttons are pressed.

Error Handling: Built-in validation for invalid mathematical expressions.

Responsive Design: Fully functional on desktop, tablet, and mobile screens using CSS Grid.

Advanced Controls: Includes functional 'Clear' (C) and 'Backspace' (←) capabilities for a better user experience.

🛠️ Tech Stack
Frontend: HTML5, CSS3

Scripting: JavaScript (ES6+)

Deployment: Optimized for hosting on platforms like Vercel or GitHub Pages.

📂 Project Structure
Plaintext
├── index.html   # Markup structure for the calculator
├── style.css    # Styling and responsive grid layout
